Cadastrando um Cliente

Uma das principais tarefas realizadas neste sistema é o cadastro de clientes. Aqui seguem informações importantes sobre o cadastro além de um vídeo exemplificando um cadastro sendo feito.

  • Apenas usuários com acesso de Gerente ou de Vendedor conseguem cadastrar Clientes
  • Os campos obrigatórios são indicados com *(asterisco)
  • Nosso sistema possui Validação de CPF, além de inputs personalizados de Telefone, data e CEP, por exemplo.
  • Nosso sistema não permite o cadastro de Clientes com o mesmo CPF
  • Quando uma UF é selecionada, serão disponibilizadas as cidades correspondentes no campo de cidades
